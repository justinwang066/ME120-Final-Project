# Physics Informed Neural Surrogates for Continuum Tumor Growth Forecasting

## Project Summary

We model tumor growth using a physics-informed neural network (PINN). A finite-difference
solver generates synthetic tumor concentration fields from a coupled reaction-diffusion PDE
system spanning 12 parameter sets (LGG and GBM regimes). A U-Net with FiLM conditioning
then learns to map an initial tumor field plus three PDE parameters (D_c, rho, alpha) to
the field at a future time, with the physics residual enforced as a training loss term.

## How To Run

```bash
python me120project.py
```

The script runs end-to-end in one shot:

1. Generates PDE training and validation datasets
2. Runs four solver validation tests and saves diagnostic plots
3. Trains the PINN for 400 epochs and saves `best_model.pt`
4. Evaluates the best checkpoint and prints per-sample metrics

No arguments or config files needed beyond what is hardcoded in the script.

## Expected Outputs

All outputs land in `pinn_output/`:

| File | Description |
|---|---|
| `val_figA_growth.png` | Growth curves and final area scatter for all simulations |
| `val_figB_fields.png` | Side-by-side initial/final fields for all 12 training sims |
| `val_figC_holdout.png` | Initial and final fields for the 3 validation hold-outs |
| `training_curves.png` | Train/val loss curves over epochs |
| `predictions.png` | PINN prediction vs. ground truth for each validation case |
| `scatter.png` | Pixel-level predicted vs. true scatter plot |
| `best_model.pt` | Best checkpoint by validation MSE |

Terminal output includes per-sim CFL checks, four solver validation test results,
per-epoch training logs, and final per-sample MSE, max error, and area error.

## Runtime Notes and Limitations

- CPU runtime is roughly 30-60 minutes depending on hardware (PDE generation dominates).
- GPU cuts training time significantly but is not required.
- The solver uses explicit forward Euler; CFL is checked automatically and will raise if violated.
- The nutrient field is approximated quasi-steadily in the PDE loss, not simulated in the network.
- The model is trained on a 12-case sweep; generalization outside the parameter ranges in
  `PARAM_SWEEP` is not guaranteed.

## Omitted Files

`tumor_project_data/` and `pinn_output/` are not included. Both directories are fully
